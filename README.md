## 🌟 Về Trang Web (About)

Đây là mã nguồn trang web cá nhân (Portfolio) của **NamGT**. Trang web được thiết kế theo phong cách Glassmorphism hiện đại, tối giản nhưng đậm chất công nghệ với rất nhiều hiệu ứng tương tác cao cấp (Custom cursor, Scroll Reveal, Particle canvas, Lo-fi ambient music).

Mục đích của trang web là nơi trưng bày các kỹ năng lập trình, dự án cá nhân và làm cầu nối liên hệ với cộng đồng mạng cũng như các nhà tuyển dụng.

### 🚀 Công nghệ sử dụng (Tech Stack)
Trang web được xây dựng hoàn toàn từ con số không (From scratch) để đảm bảo hiệu năng và tốc độ tải trang cực nhanh mà không phụ thuộc vào framework cồng kềnh:
- **Frontend:** HTML5, Vanilla CSS3 (Custom Variables, Flexbox, Grid), Vanilla JavaScript (ES6+).
- **Kỹ năng chuyên môn:** Node.js, TypeScript, Python, Lua/Luau.
- **Tính năng nổi bật:** 
  - Giao diện Sáng/Tối (Light/Dark Mode).
  - Chuyển đổi ngôn ngữ tức thời (VI/EN).
  - Trình phát nhạc Lo-fi tích hợp (Audio Context API).
  - Lớp bảo vệ chống sao chép và khóa tên miền (Domain Lock).
